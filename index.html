What is a Node?
data field "Coal" getCargo()
next field → "Timber" getNext()

Every car in the train is a node. It holds two things:
data — the value stored inside the node.
next — a reference to the next node (or null if last).

Method style
car.getCargo() car.getNext() car.setNext(x)
Direct field style
car.data car.next car.next = x
Java class definition
public class TrainCar {
  private String cargo;  // the DATA
  private TrainCar next;  // the POINTER

  public String getCargo() {
    return this.cargo;
  }

  public TrainCar getNext() {
    return this.next;
  }

  public void setNext(TrainCar n) {
    this.next = n;
  }
}
public class TrainCar {
  public String data;  // the DATA (public!)
  public TrainCar next;  // the POINTER (public!)

  // No getters/setters needed —
  // fields accessed directly

  public TrainCar(String data) {
    this.data = data;
    this.next = null;
  }
}
INSERT Adding a new node means creating a new TrainCar and wiring it in.
At the start: new car's next points to old head, then head is updated.
At the end: walk until next is null, then attach the new car.
In the middle: re-wire two pointer calls — new car points forward, previous car points to new.
DIRECT FIELD — paper style:
newCar.next = head;   head = newCar;  (insert at start)
current.next = newCar;  (insert at end after walking)
newCar.next = current.next; current.next = newCar;  (insert after)
DELETE Removing a node means making the node before it skip over it.
If deleting the head: move head to the next node.
Otherwise: walk until the node after current is the target, then re-point past it.
The removed node is now disconnected — Java's garbage collector will clean it up.
DIRECT FIELD — paper style:
head = head.next;  (delete head)
current.next = current.next.next;  (skip over target)
SEARCH Searching means walking the list one node at a time, checking each node's value.
Start at head, loop while current is not null. If the data matches — found it!
If the loop ends without a match, the value is not in the list. This is linear search — O(n) time.
DIRECT FIELD — paper style:
while (current != null) {
  if (current.data.equals("x")) return current;
  current = current.next;
}
READ DATA Reading values accesses the data field of each node as you walk the list.
The method style uses getCargo() — the getter for the private field.
The direct style reads car.data directly — simpler, used in many exam questions.
DIRECT FIELD — paper style:
System.out.println(current.data);
current = current.next;
